# Abstract Blue Strips

Minimalist abstract visual project based on color, rhythm, and motion.

## Overview

This project explores **abstract visual storytelling** using simple HTML, CSS, and optional JavaScript.
The focus is on clean composition, color gradients, and dynamic strip-based layouts.

## Live Preview

https://vud-rgb.github.io/Abstract-Blue-Strips/

## Tech Stack

* HTML5
* CSS3 (animations, gradients, layout)
* JavaScript (optional interactions)

## Project Structure

```
/index.html
/css/style.css
/js/script.js
```

## Features

* Abstract strip-based design
* Smooth visual flow
* Lightweight and fast
* Easy to customize

## How to Use

1. Clone or download the repository
2. Open `index.html` in your browser
3. Modify styles in `/css/style.css`

## Customization

You can easily change:

* Colors (gradients, backgrounds)
* Strip sizes and spacing
* Animation speed and behavior
